Lançamentos

Para fazer os lançamentos de plano de aula e notas é necessário informar a turma e a matéria, para o plano de aula também é necessário informar o mês.

Esses dados podem ser obtidos acessando o Classis.

No campo turma deve ser informado o código da turma, geralmente em um formato parecido com TI05T-1.

No campo matéria deve ser informado o nome da matéria, geralmente algo parecido com PROGRAMAÇÃO PARA INTERNET I.

No campo mês deve ser informado o mês em que será feito o lançamento do plano de aula, geralmente algo parecido com MARÇO/2014.

O mais importante a ser lembrado aqui é que o valor que você escrever em cada campo deve ser idêntico ao valor existente no Classis, qualquer letra ou símbolo diferente resultará em um erro durante a execução.

Plano de aula

O campo conteúdo do lançamento de plano de aula deve ser preenchido com uma lista de conteúdos a serem lançados.

O conteúdo de cada aula deve ficar em uma linha separada.

Por exemplo, se a matéria PROGRAMAÇÃO PARA INTERNET I possuir 30 aulas no mês MARÇO/2014, o campo conteúdo deve ser preenchido com 30 linhas, cada uma contendo um contéudo para cada aula.

Infelizmente cada mês deve ser lançado separadamente.

Notas

O campo notas do lançamento de notas deve conter em cada linha os seguintes dados: O R.A. (Registro do Aluno, Matrícula do Aluno, Nome que só a FIEMG sabe) seguido de um ou mais espaços seguidos pelas notas, separadas por espaços.

Caso um R.A. não seja encontrado ele será ignorado.

As casas decimais da nota podem ser separadas por , (vígular) ou . (ponto).

Os zeros antes do R.A. são opcionais

Exemplo 

0000191792 90,00 18,00
0000152891 89,00 51,00
0000191814 71,00 32,00
0000213333 62,00 24,00
0000216885 87,00 76,00
0000213137 75,00 97,00

Dicas

Mantenha um arquivo do Excel com os dados de notas e planos de aula.

Se você escrever uma aula por linha, no Excel, basta copiar e colar no campo para lançamento do plano de aula.

Se você utilizar as colunas "Nome do aluno", "R.A.", "Nota1", "Nota2", "NotaN...", no Excel, basta copiar e colar os dados a partir da coluna "R.A." para o lançamento de notas.
